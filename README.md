# 시대예보 — 직장도 직업도 없어질 때, 꼭 남겨야 하는 것 ☀️

송길영 『시대예보: 경량문명의 탄생』(인생질문 307회) 강연을 기반으로 만든 **15분 분량 발표 웹앱**입니다.
일기예보 컨셉의 밝은 디자인으로, **OKR Party 세미나**에서 SSD Controller 검증팀(직장인)의 AI 도입 동기부여를 목적으로 합니다.
각 슬라이드는 **송길영 작가의 핵심 워딩(인용문) + 이를 표현하는 시각화** 한 쌍으로 구성됩니다.

🔗 원본 영상: https://youtu.be/lVVPmhJjCjA

## 핵심 메시지

> **버티지 말고, AI로 무장하라.** 새로운 도구를 축복으로 받아들일지 재앙으로 경험할지는, 얼마나 빨리 받아들이느냐에 달렸다.

## 구성 (15장 · 약 15분)

1. 표지 — 2. 오늘의 예보(Agenda) — 3. 충격 질문(Hook) — 4. 변곡점 2022.11.30 —
5. 중량문명→경량문명 — 6. 변화① 지능의 범용화 — 7. 변화② 인당 시총 —
8. 경쟁의 대전환 — 9. 현장 목격담 — 10. **우리 검증팀 적용** — 11. 버티지 마세요 —
12. 축복이냐 재앙이냐(선택) — 13. 그래서 나는? — 14. 클로징 — 15. 출처

> 10번(검증팀 적용)과 13번(성찰)은 청중 맞춤을 위한 발표자 해석 슬라이드입니다.

## 발표 중 단축키

| 키 | 동작 | 키 | 동작 |
|---|---|---|---|
| `→` `Space` `PgDn` | 다음 | `←` `PgUp` | 이전 |
| `Home` `End` | 처음 / 마지막 | `N` | 발표자 노트 |
| `T` / `R` | 타이머 시작·정지 / 리셋 | `F` | 전체화면 |
| `?` | 단축키 도움말 | 모바일 | 좌우 스와이프 |

발표자 노트에는 슬라이드별 **권장 시간**과 멘트가 담겨 있습니다. 하단 ⏱️ 타이머로 15분 페이스를 관리하세요.

## 내용 수정하기

- **표지 텍스트(발표자/날짜/행사명)**: [`components/slides.tsx`](components/slides.tsx) 상단 `META` 객체
- **슬라이드 본문 / 발표자 노트**: 같은 파일의 `slides` 배열 (각 항목의 `Content`, `notes`, `pace`)
- **네비게이션·타이머·노트 UI**: [`components/Deck.tsx`](components/Deck.tsx)

## 로컬 실행

```bash
npm run build && npm run start   # 프로덕션 빌드
```
